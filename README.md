# Fake Store API Testing Project

This project demonstrates API testing using Postman.

## API under test:
https://fakestoreapi.com/

## Testing scope:
- Validate API status codes
- Verify JSON response structure
- Test positive and negative scenarios

 ## Tools used:
 - Postman
 - GitHub
 - REST API testing

 ## Endpoints tested
  - GET /products
  - GET /products/{id}
  - POST /auth/login

  ## Test artifacts
  - API Requirements
  - Test Scenarios
  - Test Cases
  - Postman Collection
  - Test Execution Report
